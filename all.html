    python 在__init__中批量设置类属性

    常见的写法：

    def __init__(self, x, y, z):
    ...
    self.x = x
    self.y = y
    self.z = z
    ...

    可以简写成如下：

    class A:
    def __init__(self, **kwargs):
        for key in kwargs:
            setattr(self, key, kwargs[key])
